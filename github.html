Currently no one has archived github.com. This webpage is about progress toward that.

I host the metadata for the repositories:
  • Full repository metadata is available in batches of 10,000 at 
    http://za3k.com/github/repos-<X>0000-<X+1>0000.json
http://za3k.com/github/repos-<X>0000-<X+1>0000.json.gz
    To download all files, run 
    
for x in {0..100}; do \
  wget "http://za3k.com/github/repos-$((x*10000))-$(((x+1)*10000)).json.gz"; \
done
    The files are around 10G compressed, 100G uncompressed.
  • You can grab greatly abbreviated metadata (recommended) as JSON.
  • Finally, you can get a txt file of just the repo names: txt.
  • I got data on github repositories using a custom tool I wrote which uses the github API.

Metadata for gists is currently unavailable from github, but I'm working with them to make it public.

Additional information:
  • githubarchive.org archives the public timeline of github.
  • Archive Team has some information about github, and you probably should talk to them if you want to archive github. We basically just don't have anyone with space.
